The Brew Budz - (Project Two)

A group of three other student and I designed this application. This was our second project. What motivated us to create this application was the love of breweries some of my team-members had for San Diego breweries. This web application focuses on locating breweries within San Diego. The users of this app can will need to create an account within our app. Once they create an account, the user can search for breweries. Once the user locates target breweries, the user then can mark the brewery the user wants by adding the brewery to his/her favorites list.

This application has other cool features as well. The information of the searched breweries consist of the address, telephone number and website of the brewery. The user can click on the address of the searched breweries and the user will be redirected to a google-maps page where the user can see the location of the brewery on the map. The user can call the brewery directly by clicking on the telephone listed for the brewery. The user can also click on the website link provided by the searched breweries and the user will be redirected to the personal website of the brewery for further information. This app will run in the broweser and feature dynamically updated Handlebars, CSS, Bootstrap, JQuery, API, AJAX, Node.js, Express.js, MySQL, Sequalize, jawsDB and MVC.
