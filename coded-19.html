Coded-19 App - (Project One)

A group of three other student and I designed this application. This was our first project. What motivated us to create this application was the COVID-19 virus that has ravaged the world in recent months. This web application focuses on finding COVID-19 cases of the States the user of this application searches, so as to better educate and ensure safety from the pandemic.

This COVID-19 Data application allows the user to see real-time updates, such as number of deaths and number of currently hospitalized people from the Coronavirus (Covid-19) effected States within the United States. This application will also allow the user to search for New York Times articles pertaining to the Covid-19. This app will run in the broweser and feature dynamically updated HTML5 and CSS, Material CSS, powered by JQuery, API, and AJAX.
